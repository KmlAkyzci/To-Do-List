# 📝 Modern To-Do List (Görev Yöneticisi)

HTML, BootStrap , JavaScript mimarisi kullanılarak geliştirilmiş; kullanıcıların günlük görevlerini planlamasını, filtrelemesini ve tarayıcı kapansa dahi verilerini korumasını sağlayan modern, minimalist ve yüksek performanslı bir görev yönetim uygulaması.

## 🚀 Canlı Demo
Uygulamanın çalışan halini tarayıcınızda anlık olarak deneyimleyin:  
👉 https://kmlakyazc-to-do-list.netlify.app/?taskName=#

---

## ✨ Öne Çıkan Özellikler
- **Kalıcı Veri (Local Storage):** Sayfa yenilense veya tarayıcı kapatılsa dahi eklenen görevlerin kaybolmaması için LocalStorage entegrasyonu.
- **Dinamik Filtreleme:** Görevleri "Hepsi", "Tamamlananlar" ve "Devam Edenler" olarak anlık süzebilme yeteneği.
